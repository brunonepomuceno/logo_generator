#instalar o PyTorch e a versão mais recente do PyTorch e garante que o backend "mps" será usado corretamente.
pip install --pre torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cpu

#Instale as bibliotecas necessárias com este comando:
pip install torch diffusers transformers accelerate safetensors

# Instala as dependências do Hugging Face e outras bibliotecas
pip install diffusers transformers accelerate safetensors Flask Flask-Cors peft

torch: A biblioteca de deep learning principal.
diffusers: A biblioteca central que gerencia o pipeline de geração.
transformers: Necessária para carregar os modelos de texto.
accelerate: Ajuda a otimizar o uso da GPU.
safetensors: Para carregar o formato de arquivo dos modelos de forma segura e rápida.
